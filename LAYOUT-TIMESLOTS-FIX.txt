  🔄 POUR TESTER

1. Redémarrer le serveur Angular :
   ng serve

2. Aller sur : http://localhost:4200/demande-demo

3. Sélectionner une date dans le calendrier

4. Observer :
   ✅ Les créneaux horaires apparaissent à DROITE du calendrier
   ✅ Scroll vertical si beaucoup de créneaux
   ✅ Grille 2 colonnes pour les heures
   ✅ Bouton "Confirmer" pleine largeur en bas

5. Tester le responsive :
   - Réduire la fenêtre < 1024px : créneaux passent en dessous
   - Réduire < 768px : tout en colonne verticale
