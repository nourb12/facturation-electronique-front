Le dark mode est maintenant automatiquement appliqué !

Toutes les variables CSS sont définies et utilisées partout :
  • Backgrounds → var(--bg-card), var(--bg-void), etc.
  • Textes → var(--tp), var(--ts), var(--tt)
  • Bordures → var(--b1), var(--b2), var(--b3)
  • Inputs → var(--input-bg), var(--input-text)

Aucune modification de code HTML nécessaire !
Les couleurs s'appliquent automatiquement via les variables CSS.
