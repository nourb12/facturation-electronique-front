🎯 POURQUOI LE DARK MODE NE S'APPLIQUE PAS ?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Le problème est que le fichier `_dark-theme-ey.scss` définit les variables
MAIS elles ne sont pas appliquées car :

1. Le cache Angular n'est pas vidé
2. Le serveur n'a pas été redémarré
3. Le cache du navigateur n'est pas vidé


🔧 SOLUTION IMMÉDIATE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ÉTAPE 1 : ARRÊTER LE SERVEUR
┌─────────────────────────────────────────────────────────────┐
│  Ctrl+C dans le terminal                                    │
└─────────────────────────────────────────────────────────────┘

ÉTAPE 2 : SUPPRIMER LE CACHE ANGULAR
┌─────────────────────────────────────────────────────────────┐
│  rm -rf .angular/cache                                      │
│  # OU sur Windows PowerShell:                               │
│  Remove-Item -Recurse -Force .angular/cache                 │
└─────────────────────────────────────────────────────────────┘

ÉTAPE 3 : REDÉMARRER LE SERVEUR
┌─────────────────────────────────────────────────────────────┐
│  ng serve                                                   │
└─────────────────────────────────────────────────────────────┘

ÉTAPE 4 : VIDER LE CACHE DU NAVIGATEUR
┌─────────────────────────────────────────────────────────────┐
│  1. Ouvrir DevTools (F12)                                   │
│  2. Clic droit sur le bouton Actualiser                     │
│  3. Choisir "Vider le cache et actualiser"                  │
└─────────────────────────────────────────────────────────────┘

🔍 COMMENT VÉRIFIER DANS LE NAVIGATEUR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. OUVRIR DEVTOOLS (F12)

2. VÉRIFIER L'ATTRIBUT data-theme
   ┌─────────────────────────────────────────────────────────────┐
   │  • Aller dans l'onglet "Elements"                           │
   │  • Sélectionner <html>                                      │
   │  • Vérifier que data-theme="dark" est présent               │
   └─────────────────────────────────────────────────────────────┘

3. VÉRIFIER LES VARIABLES CSS
   ┌─────────────────────────────────────────────────────────────┐
   │  • Dans "Elements", sélectionner <html data-theme="dark">   │
   │  • Dans le panneau "Styles", chercher :root[data-theme...]  │
   │  • Vérifier que --bg-void: #2E2E38                          │
   │  • Vérifier que --bg-card: #3A3A45                          │
   │  • Vérifier que --tp: #FFFFFF                               │
   └─────────────────────────────────────────────────────────────┘

4. VÉRIFIER LE BACKGROUND DU BODY
   ┌─────────────────────────────────────────────────────────────┐
   │  • Sélectionner <body>                                      │
   │  • Dans "Computed", chercher "background-color"             │
   │  • Devrait afficher: rgb(46, 46, 56) = #2E2E38             │
   └─────────────────────────────────────────────────────────────┘

5. VÉRIFIER UNE CARTE
   ┌─────────────────────────────────────────────────────────────┐
   │  • Inspecter n'importe quelle carte (.card, .kpi, etc.)     │
   │  • Dans "Computed", chercher "background-color"             │
   │  • Devrait afficher: rgb(58, 58, 69) = #3A3A45             │
   └─────────────────────────────────────────────────────────────┘


❌ SI LE DARK MODE NE S'APPLIQUE TOUJOURS PAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PROBLÈME 1 : data-theme="dark" n'est pas présent
┌─────────────────────────────────────────────────────────────┐
│  SOLUTION : Le ThemeService ne fonctionne pas               │
│  • Vérifier que le service est bien injecté                 │
│  • Vérifier localStorage: "ey-theme" = "dark"               │
│  • Forcer le dark mode dans le navigateur                   │
└─────────────────────────────────────────────────────────────┘

PROBLÈME 2 : Les variables CSS ne sont pas définies
┌─────────────────────────────────────────────────────────────┐
│  SOLUTION : Le fichier _dark-theme-ey.scss n'est pas chargé │
│  • Vérifier que l'import existe dans styles.scss            │
│  • Vérifier qu'il n'y a pas d'erreur de compilation SCSS    │
│  • Redémarrer le serveur avec cache vidé                    │
└─────────────────────────────────────────────────────────────┘

PROBLÈME 3 : Les couleurs sont différentes
┌─────────────────────────────────────────────────────────────┐
│  SOLUTION : Un autre fichier écrase les couleurs            │
│  • Vérifier l'ordre des imports dans styles.scss            │
│  • _dark-theme-ey.scss doit être EN PREMIER                 │
│  • Vérifier qu'il n'y a pas de !important ailleurs          │
└─────────────────────────────────────────────────────────────┘
