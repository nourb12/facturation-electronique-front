## ⚡ SOLUTION IMMÉDIATE

Si vous voulez juste appliquer le dark mode **MAINTENANT** sans lire :

```bash
# 1. Arrêter le serveur
Ctrl + C

# 2. Supprimer le cache
Remove-Item -Recurse -Force .angular/cache    # PowerShell
rm -rf .angular/cache                          # Git Bash

# 3. Redémarrer le serveur
ng serve

# 4. Vider le cache du navigateur
F12 → Clic droit sur Actualiser → Vider le cache
```

**C'est tout ! Le dark mode sera appliqué. 🎉**
