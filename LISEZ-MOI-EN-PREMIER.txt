Le dark mode EY est CORRECTEMENT CODÉ mais ne s'affiche pas car le cache Angular
contient l'ancienne version des styles.

✅ CODE : 100% CORRECT
❌ CACHE : DOIT ÊTRE VIDÉ


⚡ SOLUTION RAPIDE (2-3 MINUTES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Arrêter le serveur : Ctrl + C

2. Supprimer le cache :
   PowerShell : Remove-Item -Recurse -Force .angular/cache
   Git Bash   : rm -rf .angular/cache

3. Redémarrer le serveur : ng serve

4. Vider le cache du navigateur :
   F12 → Clic droit sur Actualiser → Vider le cache
