🐛 PROBLÈMES COURANTS ET SOLUTIONS
═══════════════════════════════════════════════════════════════════════════

Problème : "Connection refused"
Solution : Vérifier que le backend est lancé sur http://localhost:8080

Problème : "CORS error"
Solution : Ajouter la configuration CORS dans WebConfig.java

Problème : "Package not found"
Solution : flutter clean && flutter pub get

Problème : Script PowerShell bloqué
Solution : Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
